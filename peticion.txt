BUG CRÍTICO EN control/views.py — _caja_anterior_en_ciclo selecciona el 
turno incorrecto dentro del mismo día.

---

DIAGNÓSTICO CONFIRMADO

La función _caja_anterior_en_ciclo filtra con fecha__lte=fecha y ordena 
por -fecha, -creado_el. Esto incluye todos los turnos del mismo día, 
y el último turno creado (ej: Noche) siempre queda primero en el resultado.

Consecuencia: Mañana y Tarde de cada día usan el total_efectivo de Noche 
como su caja_anterior, aunque Noche aún no haya ocurrido.
Solo el último turno de cada día está correcto (al excluirse a sí mismo 
encuentra el turno anterior real).

Patrón observado en producción:
- 07/05 Mañana caja_ant=1441 → TOMADO DE: 07/05 Noche total_ef (✗)
- 07/05 Tarde  caja_ant=1441 → TOMADO DE: 07/05 Noche total_ef (✗)
- 07/05 Noche  caja_ant=1375 → TOMADO DE: 07/05 Tarde total_ef (✓ correcto)

El bug afecta todos los días desde el inicio del sistema.

FIX 1 — Agregar parámetro turno_tipo a _caja_anterior_en_ciclo

Ubicar la función _caja_anterior_en_ciclo en views.py y reemplazarla 
por esta versión corregida:

TURNO_ORDEN = {'Mañana': 0, 'Tarde': 1, 'Noche': 2}

def _caja_anterior_en_ciclo(sucursal, fecha, exclude_pk=None, turno_tipo=None):
    """
    Retorna (caja_anterior, prestamos_acum_ant).
    turno_tipo: tipo del turno actual ('Mañana', 'Tarde', 'Noche').
    Con turno_tipo, para el mismo día solo considera turnos cronológicamente anteriores.
    Sin turno_tipo, solo mira días estrictamente anteriores.
    """
    inicio_ciclo = get_inicio_ciclo(sucursal)
    if inicio_ciclo and fecha <= inicio_ciclo:
        return int(sucursal.caja_inicial or 0), 0

    qs = CuadraturaCajaDiaria.objects.filter(
        sucursal=sucursal,
    ).select_related('turno')
    if inicio_ciclo:
        qs = qs.filter(fecha__gte=inicio_ciclo)
    if exclude_pk:
        qs = qs.exclude(pk=exclude_pk)

    orden_actual = TURNO_ORDEN.get(turno_tipo, None)

    if orden_actual is not None:
        # Incluir días anteriores + turnos del mismo día con orden < actual
        turnos_anteriores = [t for t, o in TURNO_ORDEN.items() if o < orden_actual]
        from django.db.models import Q
        qs = qs.filter(
            Q(fecha__lt=fecha) |
            Q(fecha=fecha, turno__tipo_turno__in=turnos_anteriores)
        )
    else:
        # Sin turno_tipo: solo días anteriores para no mezclar mismo día
        qs = qs.filter(fecha__lt=fecha)

    prev = qs.order_by("-fecha", "-creado_el").first()
    if prev:
        return int(prev.total_efectivo or 0), int(prev.prestamos_acum or 0)
    return int(sucursal.caja_inicial or 0), 0
